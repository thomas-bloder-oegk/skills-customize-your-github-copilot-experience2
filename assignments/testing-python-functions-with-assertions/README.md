# 📘 Assignment: Testing Python Functions with Assertions

## 🎯 Objective

Learn how to check Python functions using built-in `assert` statements. You will practice writing simple functions, predicting expected results, and using tests to catch mistakes.

## 📝 Tasks

### 🛠️	Write Simple Functions to Test

#### Description
Complete a small set of Python functions that return clear, predictable results.

#### Requirements
Completed program should:

- Define a function named `is_even(number)` that returns `True` for even numbers and `False` for odd numbers.
- Define a function named `format_greeting(name)` that returns a greeting such as `Hello, Maya!`.
- Define a function named `calculate_total(price, quantity)` that returns the total cost.
- Return values from each function instead of only printing them.


### 🛠️	Add Assertion Tests

#### Description
Write `assert` statements that prove each function works for common inputs.

#### Requirements
Completed program should:

- Include at least two `assert` statements for each function.
- Test both an even and an odd number for `is_even(number)`.
- Test at least two different names for `format_greeting(name)`.
- Test at least two different price and quantity combinations for `calculate_total(price, quantity)`.
- Print a success message only after all assertions pass.


### 🛠️	Find and Fix a Failing Test

#### Description
Use one intentionally failing assertion to practice reading an error message and correcting code or expected output.

#### Requirements
Completed program should:

- Add one assertion that fails at first.
- Read the assertion error to identify what went wrong.
- Fix the function or expected value so the test passes.
- Leave the final program with all assertions passing.
